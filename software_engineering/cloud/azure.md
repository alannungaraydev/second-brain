# Productos Cloud de Azure

A continuación, se presenta una lista completa de los productos cloud ofrecidos por **Microsoft Azure**, ordenados **alfabéticamente** dentro de cada categoría. Las categorías se basan en la estructura oficial de Azure y cubren las principales áreas de servicios.

> **Nota**: Azure actualiza frecuentemente su portafolio. Verifica la información más reciente en [azure.microsoft.com/en-us/products](https://azure.microsoft.com/en-us/products/).
