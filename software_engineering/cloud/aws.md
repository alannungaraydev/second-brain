# Productos Cloud de AWS

A continuación, se presenta una lista completa de los productos cloud ofrecidos por **Amazon Web Services (AWS)**, ordenados **alfabéticamente** dentro de cada categoría. Las categorías se basan en la estructura oficial de AWS y cubren las principales áreas de servicios.

> **Nota**: AWS actualiza frecuentemente su portafolio. Verifica la información más reciente en [aws.amazon.com/products](https://aws.amazon.com/products/).

---

## Analítica (Analytics)

- **Amazon Athena** — Consulta de datos en S3 usando SQL estándar.
- **Amazon CloudWatch** — Monitoreo y observabilidad de recursos en la nube.
- **Amazon EMR** — Procesamiento de big data con Hadoop, Spark y más.
- **Amazon Kinesis** — Procesamiento de datos en streaming en tiempo real.
- **Amazon Managed Grafana** — Servicio gestionado para visualización de datos.
- **Amazon Managed Service for Prometheus** — Monitoreo de contenedores con Prometheus.
- **Amazon OpenSearch Service** — Búsqueda y análisis con OpenSearch.
- **Amazon QuickSight** — Inteligencia de negocios y visualizaciones interactivas.
- **AWS Glue** — Servicio ETL gestionado para preparación de datos.
- **AWS IoT Analytics** — Análisis de datos de dispositivos IoT.
- **AWS Lake Formation** — Gestión segura de data lakes.
- **AWS X-Ray** — Análisis de rendimiento de aplicaciones distribuidas.

---

## Aplicaciones (Application Integration)

- **Amazon AppFlow** — Transferencia de datos entre AWS y aplicaciones SaaS.
- **Amazon EventBridge** — Enrutamiento de eventos entre servicios.
- **Amazon MQ** — Broker de mensajes gestionado (ActiveMQ y RabbitMQ).
- **Amazon Simple Notification Service (SNS)** — Servicio de notificaciones pub/sub.
- **Amazon Simple Queue Service (SQS)** — Colas de mensajes gestionadas.
- **AWS Step Functions** — Orquestación de flujos de trabajo sin servidor.

---

## Computación (Compute)

- **AWS Batch** — Ejecución de trabajos por lotes a gran escala.
- **AWS Fargate** — Computación sin servidor para contenedores.
- **AWS Lambda** — Ejecución de código sin gestionar servidores.
- **Amazon EC2** — Máquinas virtuales escalables.
- **Amazon EC2 Auto Scaling** — Escalado automático de instancias EC2.
- **Amazon Lightsail** — Plataforma simplificada para aplicaciones pequeñas.
- **Elastic Container Service (ECS)** — Orquestación de contenedores.
- **Elastic Kubernetes Service (EKS)** — Kubernetes gestionado en AWS.

---

## Bases de Datos (Database)

- **Amazon Aurora** — Base de datos relacional compatible con MySQL/PostgreSQL.
- **Amazon DynamoDB** — Base de datos NoSQL de alta escalabilidad.
- **Amazon ElastiCache** — Caché en memoria (Redis y Memcached).
- **Amazon Keyspaces** — Apache Cassandra gestionada.
- **Amazon Neptune** — Base de datos de grafos.
- **Amazon QLDB** — Registro inmutable tipo libro mayor.
- **Amazon RDS** — Bases de datos relacionales gestionadas.
- **Amazon Redshift** — Data warehouse para análisis a escala petabyte.
- **Amazon Timestream** — Base de datos para series temporales.
- **AWS Database Migration Service** — Migración de bases de datos a AWS.

- **AWS CodeBuild** — Compilación y pruebas de código sin servidores.
- **AWS CodeCommit** — Repositorio Git gestionado.
- **AWS CodeDeploy** — Despliegue automatizado de aplicaciones.

- **AWS X-Ray** — Depuración y trazado de aplicaciones.
