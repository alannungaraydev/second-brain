# Productos Cloud de Google Cloud

A continuación, se presenta una lista completa de los productos cloud ofrecidos por **Google Cloud Platform (GCP)**, ordenados **alfabéticamente** dentro de cada categoría. Las categorías se basan en la estructura oficial de Google Cloud y cubren las principales áreas de servicios.

> **Nota**: Google Cloud actualiza frecuentemente su portafolio. Verifica la información más reciente en [cloud.google.com/products](https://cloud.google.com/products/).

---

## IA y Machine Learning (AI & Machine Learning)

- **AI Platform** — Plataforma unificada para construir, desplegar y escalar modelos de ML.
- **AutoML** — Crea modelos de ML personalizados con mínima experiencia en programación.
- **Cloud Natural Language** — Analiza texto para extraer sentimientos, entidades y sintaxis.
- **Cloud Speech-to-Text** — Convierte audio a texto usando reconocimiento de voz de IA.
- **Cloud Text-to-Speech** — Genera audio natural de texto con voces de IA.
- **Cloud Translation** — Traduce texto entre miles de pares de idiomas.
- **Cloud Vision AI** — Detecta objetos, caras y texto en imágenes y videos.
- **Contact Center AI** — Mejora las interacciones con clientes usando IA conversacional.
- **Document AI** — Procesa y extrae información de documentos no estructurados.
- **Gemini for Google Cloud** — Modelos de IA generativa multimodal para casos de uso empresariales.
- **Vertex AI** — Plataforma gestionada para ML que integra todo el ciclo de vida.

---

## Analítica (Analytics)

- **BigQuery** — Data warehouse serverless para analítica a escala petabyte.
- **Cloud Composer** — Orquesta flujos de trabajo de datos con Apache Airflow.
- **Cloud Data Fusion** — Plataforma de integración de datos sin código para ETL.
- **Cloud Dataplex** — Gestiona, asegura y analiza datos distribuidos en múltiples entornos.
- **Cloud Dataflow** — Servicio gestionado de procesamiento de datos por lotes y streaming.
- **Cloud Looker Studio** — Crea informes y dashboards interactivos (anteriormente Data Studio).
- **Cloud Profiler** — Perfila continuamente aplicaciones en producción para optimización.
- **Dataprep** — Prepara datos para análisis con limpieza y transformación visual.
- **Looker** — Plataforma de BI para modelado y análisis de datos semánticos.

---

## Computación (Compute)

- **Cloud Run** — Plataforma serverless para contenedores que escala automáticamente.
- **Compute Engine** — Máquinas virtuales personalizables y de alto rendimiento.
- **Google Kubernetes Engine (GKE)** — Kubernetes gestionado para contenedores.
- **Google Compute Engine VM Instances** — Instancias de VM escalables.
- **Migrate for Anthos** — Migra VMs a contenedores o VMs en Google Cloud.
- **Migrate to Virtual Machines** — Migra VMs on-premises a Compute Engine.

---

## Bases de Datos (Databases)

- **AlloyDB for PostgreSQL** — Base de datos PostgreSQL gestionada con rendimiento empresarial.
- **Bigtable** — Base de datos NoSQL de baja latencia para grandes volúmenes de datos.
- **Cloud Memorystore** — Servicios gestionados de Redis y Memcached en memoria.
- **Cloud SQL** — Bases de datos relacionales gestionadas (MySQL, PostgreSQL, SQL Server).
- **Cloud Spanner** — Base de datos relacional horizontalmente escalable y consistente.
- **Firestore** — Base de datos NoSQL document-oriented para aplicaciones móviles y web.
- **Firebase Realtime Database** — Base de datos NoSQL sincronizada en tiempo real.

---

## Contenedores y Orquestación (Containers & Orchestration)

- **Anthos** — Plataforma híbrida y multi-cloud para aplicaciones modernas.
- **Cloud Build** — Construye, prueba y despliega software en contenedores.
- **Cloud Run** — Ejecuta contenedores serverless con autoescalado.
- **Google Kubernetes Engine (GKE)** — Orquestación gestionada de Kubernetes.
- **Knative** — Plataforma serverless para Kubernetes.
- **Artifact Registry** — Almacena, gestiona y asegura imágenes de contenedores y artefactos.

---

## Desarrollo de Aplicaciones (Developer Tools)

- **Cloud Build** — CI/CD gestionado para construir pipelines.
- **Cloud Code** — Extensión para IDEs que acelera el desarrollo en la nube.
- **Cloud Shell** — Entorno de desarrollo basado en navegador con herramientas preinstaladas.
- **Cloud Workstations** — Estaciones de trabajo gestionadas para desarrollo.

---

## Híbrido y Multi-Cloud (Hybrid & Multi-Cloud)

- **Anthos** — Gestiona aplicaciones Kubernetes en cualquier infraestructura.
- **Anthos Service Mesh** — Conectividad segura y observabilidad para microservicios.
- **Anthos on bare metal** — Ejecuta Anthos en hardware dedicado on-premises.
- **Assured Workloads** — Cumple regulaciones con aislamiento de datos.
- **Cloud Interconnect** — Conexiones dedicadas de alta velocidad a Google Cloud.
- **Connected Sheet** — Analiza datos de BigQuery directamente en Google Sheets.

---

## Internet de las Cosas (IoT)

- **Cloud IoT Core** — Gestiona dispositivos IoT a escala con conectividad segura.

---

## Red y Entrega de Contenido (Networking & Content Delivery)

- **CDN** — Entrega de contenido global con baja latencia.
- **Cloud CDN** — Red de distribución de contenido integrada con HTTP(S) Load Balancing.
- **Cloud DNS** — Servicio de DNS autoritativo y escalable.
- **Cloud Load Balancing** — Balanceo de carga global para tráfico HTTP(S), TCP/SSL y UDP.
- **Cloud Router** — Interconexión de redes VPC con BGP dinámico.
- **Cloud VPN** — Conexiones VPN seguras entre redes on-premises y VPC.
- **Network Connectivity Center** — Gestiona conexiones híbridas.
- **Network Intelligence Center** — Monitorea y optimiza la red.
- **Premium Network Tier** — Red global de baja latencia.
- **Standard Network Tier** — Red regional con rutas de Google.
- **Virtual Private Cloud (VPC)** — Redes privadas aisladas y escalables.

---

## Plataformas (Platforms)

- **App Engine** — Plataforma PaaS para aplicaciones web escalables.
- **Firebase** — Plataforma para desarrollo de aplicaciones móviles y web.
- **Apigee** — Plataforma de API management para monetizar y asegurar APIs.

---

## Seguridad y Cumplimiento (Security & Compliance)

- **Access Context Manager** — Define contextos de seguridad basados en señales de usuario y dispositivo.
- **Binary Authorization** — Verifica la integridad de contenedores en despliegues.
- **Certificate Authority Service** — Emite certificados PKI privados.
- **Cloud Data Loss Prevention** — Descubre, clasifica y protege datos sensibles.
- **Cloud IAM** — Gestiona acceso a recursos con políticas granulares.
- **Identity-Aware Proxy** — Controla acceso a aplicaciones sin VPN.
- **Security Command Center** — Gestiona la postura de seguridad y cumplimiento.
- **VPC Service Controls** — Protege datos contra fugas en entornos de alto riesgo.

---

## Almacenamiento (Storage)

- **Cloud Storage** — Almacenamiento de objetos escalable y duradero.
- **Cloud Filestore** — Sistema de archivos NFS gestionado de alto rendimiento.
- **Persistent Disk** — Almacenamiento de bloques para Compute Engine.
- **Transfer Appliance** — Dispositivo físico para transferencias de datos offline.

---

## Otros Servicios (Other)

- **Chrome Enterprise Premium** — Seguridad y gestión para navegadores Chrome.
- **Cloud Billing** — Gestiona costos y presupuestos.
- **Cloud Monitoring** — Monitorea el rendimiento de aplicaciones y infraestructura.
- **Cloud Operations Suite** — Suite para observabilidad (incluye Logging, Monitoring, Trace).
- **Cloud Trace** — Rastrea latencia en aplicaciones distribuidas.
- **Operations Suite** — Herramientas para operaciones y confiabilidad del sitio.
