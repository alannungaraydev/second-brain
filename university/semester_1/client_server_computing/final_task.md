For the final task, a ticketing system will be created for free events in which an event will be selected.
When the user selects the event, they will be able to fill in their personal data and they will be registered.
The administrator user will be able to register, view, edit and delete an event.
The administrator user will be able to register, view, edit and delete moderator users.
The moderator user will be able to register tickets for an event until it is full or the event start time has arrived.
