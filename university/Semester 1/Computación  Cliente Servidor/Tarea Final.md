Para la tarea final se realizará un sistema de boletaje para eventos gratuitos en los cuales se escogerá un evento.
Al usuario escoger el evento podrá llenar sus datos personales y quedarán registrados.
El usuario administrador podrá registrar ver editar y dar de baja un evento.
El usuario administrador podrá registrar ver editar y dar de baja a los usuarios moderadores.
El usuario moderador  podrá registrar boletos a un evento hasta que este se encuentre lleno o sea la hora iniciada del evento.
