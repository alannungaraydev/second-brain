# Tailwind CSS

A library with utility-first CSS framework for rapidly building custom user interfaces.

!> Is highly recommendable refresh knowledge with [Modern CSS Document](./styles/modern_css.md)

To get started with Tailwind CSS, first you should understand its core concepts:

- Utility-First: Tailwind provides low-level utility classes that let you build complex designs without writing custom CSS.
- Responsive Design: Tailwind includes responsive variants for all utilities, allowing you to create designs that
- Naming Conventions: Tailwind uses a consistent naming convention for its classes, making it easy to understand and use.
